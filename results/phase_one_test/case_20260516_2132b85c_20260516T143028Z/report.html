How To Read This Report

这份报告的目的不是展示漂亮结果，而是帮你定位项目问题。

Current Evidence Atomizer generates Evidence Atoms by LLM directly from items, attributes, clinical assertions, and EvidenceEventFrame nodes. It does not use coverage units and does not run dedup.

阅读顺序建议：先看 End-to-end Flow，再看 Stage Diagnostics，然后看 Item Lineage Matrix。最后到 Problem Dashboard 找重复出现的 warning/error code。

Stage Diagnostics

Input Loader

作用：把你选择的病例文件变成一个字符串。目的：后面所有对象都必须能追溯到这段原文。

输入

data/01.txt (1027 chars)

输出

raw_text + selected_file payload; input_order=1

Schema

selected_file.json 是测试脚本自定义 payload；RawTextInput 在 Case Structurer 里生成。

调用机制

文件扫描、UTF-8 读取、记录路径/字节数/字符数。

看问题时重点盯

如果原文切分或编码不稳定，后面 source_span 全都会偏。

Case Structurer

作用：从原文里切出 section 和 item。目的：把长文本变成后续模块能逐条处理的病例事实。

输入

raw_text

输出

8 sections, 12 structured items

Schema

RawTextInput, StageContext, ClinicalSection, StructuredClinicalItem, SourceSpan, CaseStructuringResult

调用机制

RawInputBuilder -> StageContextExtractor -> ClinicalSectionExtractor -> SectionNormalizer -> StructuredClinicalItemExtractor -> ItemNormalizer -> SourceSpanResolver -> CaseStructuringAssembler

看问题时重点盯

item 边界太粗会让后面 assertion/frame 变复杂；item 边界太碎会丢上下文。

Source Span Correction

作用：确认每个抽出的对象真的来自原文。目的：防止 LLM 编造或 span 坐标错位。

输入

initial CaseStructuringResult

输出

initial issues=1, corrections=1, final issues=0

Schema

SourceSpanValidationReport, SourceSpanCorrectionReport, SourceSpanValidationCorrectionResult

调用机制

StrictSourceSpanValidator + DeterministicSourceSpanCorrector + validate_and_correct_source_spans

看问题时重点盯

如果 correction 很多，说明上游抽取的 span grounding 不稳。

Attribute Extractor

作用：在每个 item 里找属性 span。目的：把“8年”“2片/次”“2024年6月11日”这类修饰信息结构化。

输入

调用机制

AttributeSpanRoleExtractor -> AttributeSpanValidator -> RoleConstraintValidator -> AttributeNormalizer -> AttributeAssembler -> AttributeExtractionValidator

attribute_role 或 applies_to_text 错，会影响 frame 节点上挂载的属性来源。

Clinical Assertion Resolver

作用：把 item 内的临床对象和肯否定状态拆出来。目的：为树形证据提供最小对象节点。

Schema

EvidenceAtom, ItemEvidenceLink, EvidenceAtomizationResult, EvidenceAtomizationValidationReport
